# Fibonacci Calculator

## Description

This project is a simple command-line application that calculates the Fibonacci number for a given input. The application prompts the user to enter a number and then computes the corresponding Fibonacci value using an iterative approach.

### Motivation

The motivation behind this project is to demonstrate basic Rust programming concepts, including user input handling, loops, and function definitions. It serves as a practical example for beginners learning Rust.

### Problem Solved

This application provides a straightforward way to calculate Fibonacci numbers, which are commonly used in various mathematical and computational applications.

### What I Learned

Through this project, I learned about:
- Handling user input in Rust.
- Implementing loops and conditionals.
- Writing functions and managing variable scopes.

## Installation

To run this project, ensure you have Rust installed on your machine. You can install Rust by following the instructions at [rust-lang.org](https://www.rust-lang.org/tools/install).

Clone the repository:

`bash
git clone https://github.com/oscar-dev19/rusty_fibonacci.git
`

## Usage

To run the application, use the following command:

`bash
cargo run
`


You will be prompted to enter a number. The application will then display the Fibonacci number for the entered value.
